## 🎨 ВИЗУАЛНИ ПОДОБРЕНИЯ

### 6. ТИПОГРАФИЯ

#### ⚠️ ПРОБЛЕМИ В СНИМКАТА:

**6.1. ТЕКСТЪТ Е ТВЪРДЕ МАЛЪК**
```
ТЕКУЩО: Стандартен размер fonts
ПРОБЛЕМ: На тъмен фон трябва по-големи текстове

ПРЕПОРЪКИ:
- H1 (Hero): 56-64px, bold
- H2 (Sections): 36-42px, semi-bold  
- Body: 18-20px (не 16px!)
- Small text: 14-16px

Font family: Inter или Manrope (съвременни, четливи)
```

**6.2. BUTTON HIERARCHY**
```
ТРЯБВА ясна йерархия:

Primary CTA (Започни сега):
- Background: #10B981 (green)
- Larger size: 48px height
- Bold text
- Shadow для depth

Secondary (Научи повече):
- Ghost button (outline)
- Lighter text
- No background

Tertiary (линкове):
- Underline on hover
- Subtle color
```

### 7. SPACING & LAYOUT

**7.1. ЛИПСВА BREATHING ROOM**
```
ПРОБЛЕМ: Калкулаторът изглежда тесен

РЕШЕНИЕ:
- Увеличи padding: 80-120px vertical
- Container max-width: 1200px
- Side margins: 24px minimum
- Section spacing: 120-160px между секции
```

**7.2. GRID SYSTEM**
```
Използвай responsive grid:

Desktop (1200px+):  12 колони
Tablet (768-1199):  8 колони
Mobile (<768px):    4 колони

Калкулаторът да е:
- Desktop: 8 колони wide, centered
- Mobile: Full width с padding
```
